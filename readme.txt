1. aplikasi dimulai dari file: FrmPenjualan.py
2. file FrmDetail.py dipanggil setelah FrmPenjualan telah berhasil menyimpan data  
   dan digunakan untuk menambahkan data detail penjualan
3. Setiap entry detail penjualan akan meng update total bayar di tabel penjualan
4. file FrmViewDetail dipanggil melalui tombol View Detail ketika FrmPenjualan memanggil 
   data yang telah ada sebelumnya
5. Proses set Lunas bisa dilakukan pada FrmDetail ataupun FrmViewDetail
